Shift all notes up or down by semitones

Transpose

Transpose shifts all notes up or down by a specified number of semitones. This feature is controlled via button actions, not configuration.

How It Works

Transpose is a runtime feature managed by the Actions system:

  • Toggle on/off using button actions
  • When active, all notes are shifted by the specified number of semitones
  • State is maintained internally by the Actions class
  • Does not require configuration sections

Configuring Transpose Actions

Map transpose actions to buttons using action_rules:

Toggle Transpose (On/Off)

{
  "action_rules": {
    "rules": [
      {
        "id": "toggle-transpose",
        "name": "Toggle Transpose",
        "button": "button:primary",
        "action": ["toggle-transpose", 12],
        "trigger": "press"
      }
    ]
  }
}

Parameters:

  • First parameter: semitones to transpose (default: 12)

Common Transpose Values

Semitones Interval Description
12 Octave up Most common use case
-12 Octave down Lower register
7 Perfect fifth up Harmonic shift
-7 Perfect fifth down Harmonic shift
5 Perfect fourth up Harmonic shift
-5 Perfect fourth down Harmonic shift

Example: Multiple Transpose Buttons

{
  "action_rules": {
    "rules": [
      {
        "id": "octave-up",
        "name": "Octave Up",
        "button": "button:1",
        "action": ["toggle-transpose", 12],
        "trigger": "press"
      },
      {
        "id": "octave-down",
        "name": "Octave Down",
        "button": "button:2",
        "action": ["toggle-transpose", -12],
        "trigger": "press"
      },
      {
        "id": "fifth-up",
        "name": "Fifth Up",
        "button": "button:3",
        "action": ["toggle-transpose", 7],
        "trigger": "press"
      }
    ]
  }
}

Direct Transpose Action

Use the transpose action to set a specific transpose value without toggling:

{
  "action_rules": {
    "rules": [
      {
        "id": "set-transpose",
        "name": "Set Transpose to +12",
        "button": "button:primary",
        "action": ["transpose", 12],
        "trigger": "press"
      }
    ]
  }
}

Startup Rules

Set transpose to be active on startup:

{
  "action_rules": {
    "startup_rules": [
      {
        "id": "startup-transpose",
        "name": "Enable Transpose on Startup",
        "action": ["toggle-transpose", 12]
      }
    ]
  }
}
