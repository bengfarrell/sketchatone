Create tremolo effects with automatic note repetition

Note Repeater

The note repeater automatically re-triggers notes while you hold the pen down, creating tremolo or strumming effects. This feature is controlled via button actions, not configuration.

How It Works

The note repeater is a runtime feature managed by the Actions system:

  • Toggle on/off using button actions
  • When active, notes are automatically re-triggered while pen is pressed
  • Repeat rate is controlled by pen pressure
  • State is maintained internally by the Actions class
  • Does not require configuration sections

Behavior:

  1. Triggers notes when you first touch the tablet
  2. Continues re-triggering notes at a rate controlled by pressure
  3. Stops when you lift the pen

Higher pressure = faster repetition.

Configuring Repeater Actions

Map repeater actions to buttons using action_rules:

Toggle Repeater (On/Off)

{
  "action_rules": {
    "rules": [
      {
        "id": "toggle-repeater",
        "name": "Toggle Note Repeater",
        "button": "button:secondary",
        "action": ["toggle-repeater", 2.0, 1.5],
        "trigger": "press"
      }
    ]
  }
}

Parameters:

  • First parameter: pressureMultiplier (default: 1.0) - How much pressure affects repeat speed
  • Second parameter: frequencyMultiplier (default: 1.0) - Base repeat frequency multiplier

Example Configurations

Gentle Tremolo

{
  "action_rules": {
    "rules": [
      {
        "id": "gentle-tremolo",
        "name": "Gentle Tremolo",
        "button": "button:1",
        "action": ["toggle-repeater", 0.5, 0.5],
        "trigger": "press"
      }
    ]
  }
}

Aggressive Strumming

{
  "action_rules": {
    "rules": [

        "id": "aggressive-strum",

        "button": "button:2",
        "action": ["toggle-repeater", 2.0, 1.5],
        "trigger": "press"

    ]
  }
}

Startup Rules

Enable repeater on startup:

{
  "action_rules": {
    "startup_rules": [
      {
        "id": "startup-repeater",
        "name": "Enable Repeater on Startup",
        "action": ["toggle-repeater", 1.0, 1.0]
      }
    ]
  }
}

Parameter Guide

Pressure Multiplier

Controls how much pen pressure affects repeat speed:

  • 0.5 - Gentle, less pressure-sensitive
  • 1.0 - Default, moderate pressure response
  • 2.0 - Aggressive, very pressure-sensitive

Frequency Multiplier

Controls the base repeat frequency:

  • 0.5 - Slow tremolo
  • 1.0 - Default speed
  • 1.5 - Fast strumming
  • 2.0 - Very fast repetition
