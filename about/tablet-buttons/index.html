Map tablet hardware buttons to chord progressions

Tablet Buttons

Many graphics tablets have hardware buttons along the edge. Sketchatone can map these to chord changes, letting you switch chords while playing.

Button IDs

Tablet buttons are identified as:

  • button:1 - First tablet hardware button
  • button:2 - Second tablet hardware button
  • button:3 - Third tablet hardware button
  • button:N - Nth tablet hardware button

Quick Example: Chord Progression

{
  "action_rules": {
    "button_names": {
      "button:1": "C Major",
      "button:2": "G Major",
      "button:3": "A Minor",
      "button:4": "F Major"
    },
    "groups": [
      {
        "id": "chord-buttons",
        "name": "Chord Buttons",
        "buttons": ["button:1", "button:2", "button:3", "button:4"]
      }
    ],
    "group_rules": [
      {
        "id": "chord-progression-rule",
        "name": "C Major Pop Progression",
        "group_id": "chord-buttons",
        "trigger": "press",
        "action": {
          "type": "chord-progression",
          "progression": "c-major-pop",
          "octave": 4
        }
      }
    ]
  }
}

This maps:

  • Button 1 → C major
  • Button 2 → G major
  • Button 3 → A minor
  • Button 4 → F major

Available Chord Progressions

Progression Chords Description
c-major-pop C, G, Am, F I-V-vi-IV in C major
c-major-50s C, Am, F, G I-vi-IV-V in C major
g-major-pop G, D, Em, C I-V-vi-IV in G major
d-major-pop D, A, Bm, G I-V-vi-IV in D major
a-major-pop A, E, F#m, D I-V-vi-IV in A major
a-minor-pop Am, F, C, G i-VI-III-VII in A minor
e-minor-pop Em, C, G, D i-VI-III-VII in E minor

Individual Button Actions

You can also map individual buttons to specific chords:

{
  "action_rules": {
    "rules": [
      {
        "id": "button-1-chord",
        "name": "Set C Major",
        "button": "button:1",
        "action": ["set-chord", "C", 4],
        "trigger": "press"
      },
      {
        "id": "button-2-chord",
        "name": "Set G Major",
        "button": "button:2",
        "action": ["set-chord", "G", 4],
        "trigger": "press"
      }
    ]
  }
}

Other Actions

Tablet buttons can be mapped to any action, not just chords:

{
  "action_rules": {
    "rules": [
      {
        "id": "button-1-transpose",
        "name": "Octave Up",
        "button": "button:1",
        "action": ["toggle-transpose", 12],
        "trigger": "press"
      },
      {
        "id": "button-2-repeater",
        "name": "Toggle Repeater",
        "button": "button:2",
        "action": ["toggle-repeater", 2.0, 1.5],
        "trigger": "press"
      }
    ]
  }
}
