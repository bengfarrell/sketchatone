Map pen buttons to actions

Stylus Buttons

Most graphics tablet pens have one or two buttons on the barrel. Sketchatone can map these to various actions like transpose, note repeater, and more.

Button IDs

Stylus buttons are identified as:

  • button:primary - Primary stylus button (barrel button 1)
  • button:secondary - Secondary stylus button (barrel button 2)

Quick Example

{
  "action_rules": {
    "button_names": {
      "button:primary": "Primary Stylus",
      "button:secondary": "Secondary Stylus"
    },
    "rules": [
      {
        "id": "stylus-primary-transpose",
        "name": "Toggle Transpose",
        "button": "button:primary",
        "action": ["toggle-transpose", 12],
        "trigger": "press"
      },
      {
        "id": "stylus-secondary-repeater",
        "name": "Toggle Repeater",
        "button": "button:secondary",
        "action": ["toggle-repeater", 2.0, 1.5],
        "trigger": "press"
      }
    ]
  }
}

This configuration:

  • Maps primary button to toggle transpose (octave up)
  • Maps secondary button to toggle note repeater

Common Actions for Stylus Buttons

Action Description Example
toggle-transpose Toggle transpose on/off ["toggle-transpose", 12]
toggle-repeater Toggle note repeater ["toggle-repeater", 2.0, 1.5]
set-chord Set specific chord ["set-chord", "Am", 4]

Trigger Types

Trigger Description
press Action fires when button is pressed
release Action fires when button is released (default)
hold Action fires while button is held
